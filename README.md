# Satyendra Kumar Yadav — Portfolio

Premium portfolio website for a Warehouse Operations / Supply Chain Team Leader.

## Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Lenis smooth scroll
- Lucide + React Icons


```bash
npm run dev
npm run build
npm start
npm run lint
```
