# Satyendra Kumar Yadav — Portfolio

Premium portfolio website for a Warehouse Operations / Supply Chain Team Leader.

## Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Lenis smooth scroll
- Lucide + React Icons

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- Content: `src/constants/data.ts`
- Certificates: add entries to `CERTIFICATES` and place images in `public/certificates/`
- Resume: replace `public/resume.pdf`
- Portrait: replace the hero placeholder in `src/sections/Hero.tsx`
- Contact details: update `CONTACT` in `src/constants/data.ts`

## Scripts

```bash
npm run dev
npm run build
npm start
npm run lint
```
